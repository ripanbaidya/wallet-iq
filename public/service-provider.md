# ⚙️ Service Providers

This project leverages a combination of modern cloud services and tools for development, scalability, and production deployment.

---

## 🗄️ Database

* 🔹 **PostgreSQL (Docker Compose)** — Used for local development
* 🔹 **PostgreSQL (Neon - Free Tier)** — Used in production for serverless database hosting

🔗 [https://console.neon.tech/app](https://console.neon.tech/app)

---

## ⚡ Caching

* 🔹 **Redis (Docker Compose)** — Used for local development
* 🔹 **Redis (Upstash)** — Used in production for serverless caching

🔗 [https://console.upstash.com/redis](https://console.upstash.com/redis)

---

## 📧 Mail Service

* 🔹 **Gmail SMTP** — Used for sending emails in both local and production environments

---

## 💳 Payment Gateway

* 🔹 **Razorpay (Test Mode)** — Integrated for handling payments in both local and production (test environment)

🔗 [https://razorpay.com/docs/payment-gateway/dashboard/test-mode/](https://razorpay.com/docs/payment-gateway/dashboard/test-mode/)

---

## 🌐 Frontend Deployment

* 🔹 **Vercel** — Used for hosting and deploying the frontend

🔗 [https://vercel.com](https://vercel.com)

---

## 🚀 Backend Deployment

* 🔹 **VPS (Docker + Nginx + SSL)** — Backend is deployed on a Virtual Private Server using containerized services with secure HTTPS setup
